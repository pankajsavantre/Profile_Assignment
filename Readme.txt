RESTAPI- url (Secured using spring security)
================================================================================
Signup User-- http://localhost:8080/api/add     ---- No need to login
Getting User -- http://localhost:8080/api/get      --- Required to login
Modifying User-- http://localhost:8080/api/update/{id}   //Ex: http://localhost:8080/api/update/1 --- Required to login to access this
Delete User --http://localhost:8080/api/delete/{id}      //Ex: http://localhost:8080/api/delete/1 -- Required to login to access this

------------------------------------------------------------------------------------

Database Used : MYSQL
Database Name : Profile  //Change name from application.properties file
